# 格式一：简历呈现版 (The Resume)

*   **用途**：写在简历上，给 HR 和筛选系统看。
*   **特点**：精简、结果导向、寸土寸金。
*   **前置要求**：请先阅读 [通用基线](./project-experience-baseline.md)，严格遵守 STAR 结构和语言避坑规则。

## 1. 编写模板

```text
【项目名称】 | 时间跨度
项目简介：B2C 电商平台，日均订单 5W+，100W+ DAU
技术栈：Vue 3 + TypeScript + Vite + Pinia
职责与成果：
- 重构商品详情页，引入虚拟滚动与懒加载，首屏时间从 3.2s 降至 1.1s
- 封装 15+ 通用业务组件（筛选器/购物车），新页面开发效率提升 50%
- 接入 Sentry 监控体系，捕获并修复 Top 5 异常，Crash 率降低 20%
```

## 2. 检查清单

*   [ ] **篇幅**：控制在 3-5 条成果，每条 1-2 行。
*   [ ] **排序**：按时间倒序排列，精选 3-4 个核心项目。
*   [ ] **内容**：无废话，无自我评价，全是干货数据。
