## 一句话定位

在模板列表点击"开始使用"时，弹出公司选择弹窗，用户选择公司后调用模板使用接口，实现模板与公司的关联使用。

## 核心目标

1. **弹窗交互**：点击"开始使用"按钮时弹出公司选择弹窗
2. **公司搜索**：复用 `CorpPresearch` 组件实现公司搜索与选择
3. **API 调用**：选择公司后调用 `report/template/use` 接口
4. **用户体验**：流畅的交互流程，清晰的错误提示
