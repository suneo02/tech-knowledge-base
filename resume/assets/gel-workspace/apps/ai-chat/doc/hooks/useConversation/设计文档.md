# useConversation Hook

## 概述
`useConversation` 是一个自定义 Hook，用于管理聊天应用中的会话列表状态和相关操作。它提供了创建新会话、切换会话等基本功能。

## 详细功能说明
Hook 实现了以下核心功能：
1. 管理会话列表状态
2. 维护当前活动会话
3. 提供新增会话的方法
4. 提供会话切换的方法

## 参数说明
Hook 不接收任何参数。

## 返回值说明
返回一个对象，包含以下属性：
- `conversationsItems: ConversationItem[]` - 会话列表数据
- `activeKey: string` - 当前活动会话的 key
- `onAddConversation: () => void` - 添加新会话的方法
- `onConversationClick: (key: string) => void` - 切换会话的方法

## 使用示例
```typescript
const ChatComponent: React.FC = () => {
  const { 
    conversationsItems, 
    activeKey, 
    onAddConversation, 
    onConversationClick 
  } = useConversation()

  return (
    <div>
      <button onClick={onAddConversation}>New Chat</button>
      <ConversationList
        items={conversationsItems}
        activeKey={activeKey}
        onActiveChange={onConversationClick}
      />
    </div>
  )
}
