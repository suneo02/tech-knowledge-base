# SmartFillModal 组件设计文档

## 概述

SmartFillModal 是一个用于AI生成列表格数据的弹窗组件，提供了模型选择、工具配置、提示语编辑等功能，帮助用户通过AI模型自动填充表格数据。

## 详细功能说明

SmartFillModal 组件基于 Ant Design 的 Modal 组件封装，主要包含以下功能区域：

1. **模板查看**：提供查看模板的功能，通过 Carousel 组件实现轮播效果
2. **模型选择**：允许用户选择不同的AI模型进行AI生成列
3. **工具配置**：提供工具使用开关，可以根据链接获取相关信息
4. **提示语编辑**：通过 Mentions 组件实现引用表格列的功能，帮助用户编写更精准的提示语
5. **自动更新设置**：提供自动更新开关，并显示相关提示信息

组件内部使用 Carousel 实现了分页效果，可以在不同的配置页面之间切换。

## 参数说明

| 参数名   | 类型                   | 描述                                   |
| -------- | ---------------------- | -------------------------------------- |
| open     | boolean                | 控制Modal是否显示                      |
| onCancel | () => void             | 关闭Modal的回调函数                    |
| onOk     | () => void             | 确认按钮的回调函数                     |
| columns  | ExtendedColumnDefine[] | 表格列配置，用于生成提示语中的引用选项 |

## 返回值说明

该组件无返回值，是一个UI组件。

## 使用示例

```tsx
import { useState } from 'react'
import { Button } from 'antd'
import { SmartFillModal } from '@/components/modal'

const MyComponent = () => {
  const [modalOpen, setModalOpen] = useState(false)
  const columns = [
    { field: 'name', title: '姓名', width: 100 },
    { field: 'age', title: '年龄', width: 100 },
    { field: 'address', title: '地址', width: 200 },
  ]

  return (
    <>
      <Button onClick={() => setModalOpen(true)}>打开AI生成列</Button>
      <SmartFillModal
        open={modalOpen}
        onOk={() => setModalOpen(false)}
        onCancel={() => setModalOpen(false)}
        columns={columns}
      />
    </>
  )
}
```
