# 🇩🇪 学业风险深度解析 (Academic Risks Deep Dive)

> **文档定位：** 德国留学 "Hard Mode" 的生存手册。
> **核心原则：** 战略上藐视（你比他们有经验），战术上重视（规则比你想象的死板）。

---

## 1. 考试规则：三次不过即死亡 (The "Drittversuch" Rule)

这是德国大学最令人闻风丧胆的规则。

### 1.1 规则详情 [1]
*   **基本原则:** 任何一门必修课 (Pflichtmodul) 或选修课 (Wahlpflichtmodul)，你通常只有 **3 次** 考试机会。
*   **计数方式:**
    *   Attempt 1: 期末考试 (Klausur)。挂科 (5.0)。
    *   Attempt 2: 补考 (Nachklausur)。通常在次学期初。再挂。
    *   **Attempt 3 (Letztversuch):** “最后一搏”。
*   **后果:** 第 3 次挂科 = **Exmatrikulation (强制退学)**。
    *   **连带伤害:** 你将被**终身禁止**在德国任何公立大学攻读该专业 (Computer Science)。你的留学梦直接碎裂。

### 1.2 避坑指南 (Mitigation)
*   **Härtefallantrag (困难申请):** 虽然有“第四次机会” (Härtefall) 的传说，但这通常仅限于极端的个人变故（如亲人离世、重病）。不要把希望寄托在这上面。
*   **Mündliche Prüfung (口试):** 很多 FH 的考规 (Prüfungsordnung) 允许第 3 次笔试改为口试。
    *   *策略:* 申请口试！只要你不是一问三不知，教授通常会给你一个 4.0 (及格)，让你通过。
*   **Attest (病假条) 战术:** 如果你在第 3 次考试前觉得复习不到位，**绝对不要去考场**。去医生那里开病假条 (Krankmeldung)，合法推迟考试。

## 2. 考试注册陷阱 (Exam Registration) [4]

*   **陷阱:** 在德国，**选课 (Course Registration) ≠ 考试注册 (Exam Registration)**。
*   **截止日期:** 每个学期都有一个严格的考试注册窗口期 (Anmeldezeitraum)。
*   **后果:**
    *   **忘注册:** 你不能参加考试。
    *   **忘退考 (Abmeldung):** 如果你注册了但没去考（且没病假条），系统自动记为 **5.0 (挂科)**。这会消耗你的一次宝贵尝试机会！
*   **对策:** 在日历上设置 3 个提醒。

## 3. FH 毕业论文 (Thesis)

虽然 FH 课程简单，但毕业论文 (Bachelor-/Masterarbeit) 依然严格。

*   **查重率:** 德国对学术诚信极度敏感（多位政客因抄袭下台）。AI 生成的内容 (ChatGPT) 如果不标注，被发现直接判定不及格。
*   **企业合作:** 你的优势在于可以在公司写论文 (Industry Thesis)。
    *   *利:* 有工资，题目实际。
    *   *弊:* 公司想保密 (NDA)，教授想发表。你需要协调好双方的利益。

---

## 4. 小组作业 (Group Projects) [3]

FH 大量课程是小组作业。

*   **风险:** 遇到“摆烂”队友 (Free riders)。德国教授通常不管组内纠纷，最后全组低分。
*   **对策:**
    *   第一节课主动找那些看起来靠谱的（比如前排记笔记的、或者也是大龄留学生的）组队。
    *   保留所有 Git Commit 记录，证明你的贡献。
