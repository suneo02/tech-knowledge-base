# 🇨🇦 加拿大 (Canada)：最后的北美堡垒

> **核心评级：** 🌟🌟🌟🌟 (Buy)
> **适合人群：** 想要绝对身份安全感、把加拿大当跳板去美国、能忍受寒冷、英语好。

## 1. 为什么加拿大是你的“保底王”？

加拿大是西方七国集团 (G7) 中移民政策最透明、最友好的国家。
*   **对于你：** 只要分数够，不需要雇主担保（不像欧洲），直接拿枫叶卡登陆。这意味着你一落地就是自由人，想去哪工作去哪工作，不用看老板脸色。
*   **美国跳板：** 拿到加拿大护照（需住满 3 年）后，可以通过 **TN Visa** 随时去美国工作，薪资翻倍。这是很多华人码农的终极路线：*CN -> CA -> US*。

## 2. 移民路径：Express Entry (EE) 实战模拟

这是加拿大联邦技术移民的核心，不拼爹，拼分数 (CRS)。

### 2.1 CRS 打分模拟 (假设场景)
基于你的背景：**24岁，单身，本科学历，1年经验，雅思8777**。

| 打分项 | 你的得分 (预估) | 分析 |
| :--- | :--- | :--- |
| **年龄** | 110 (满分) | 29 岁以下都是满分，这是你最大的资本。 |
| **学历** | 120 | 本科学位。如果能读个双学位/硕士，还能加分。 |
| **语言 (CLB 9)** | 124 | 假设雅思 听8 说7 读7 写7。如果有一个 6.5，分数会掉一大截。 |
| **工作经验** | 0 - 13 | **痛点：** 联邦技术移民 (FSW) 要求至少 1 年连续全职。但只有 1 年经验分很低。 |
| **交叉加分** | 13 - 25 | 学历+语言的组合奖励分。 |
| **总分 (Total)** | **~380 - 410** | **危险区**。目前 EE 全类别邀请分通常在 500+。 |

### 2.2 破局之道：STEM 定向抽取与省提名
既然常规分不够，怎么破？

1.  **STEM Category-Based Selection (分类定向抽取):**
    *   2023 年起，加拿大优先抽取 STEM 职业（包括 Software Developers）。
    *   **分数线：** STEM 的分数线通常比全类别低 20-40 分（约 480 左右）。
    *   *现状：* 哪怕是 STEM，你目前的 400 分出头还是不够。你需要 **3 年经验** 才能把分数冲上去。

2.  **OINP (安省省提名) Tech Draw:**
    *   安大略省（多伦多所在省）会从池子里捞 IT 人才。分数线通常在 460-480。
    *   *策略：* 依然需要更多经验或更高学历。

3.  **最终大招：留学转移民 (Study Permit):**
    *   去加拿大读个 1-2 年的 Master (如 BC 省的 NYIT, NEU 分校)。
    *   **BC 省硕博通道 (BCPNP Tech):** 毕业即省提名，无需工作 Offer。
    *   这是目前最稳的一条路，但成本较高（学费+生活费约 40-50w RMB）。

## 3. 钱袋子分析 (The Money)

### 3.1 薪资水平
*   **Junior SDE:** CAD 70k - 90k (≈36-47w RMB)
*   **Intermediate SDE:** CAD 90k - 120k (≈47-63w RMB)
*   **Senior SDE:** CAD 130k+ (≈68w RMB+)

### 3.2 真实生活成本 (多伦多 Toronto 为例)
*注：多伦多和温哥华是加拿大最贵的城市。*

| 开销项目 | 费用估算 (CAD/月) | 真实体验 |
| :--- | :--- | :--- |
| **房租 (1BR Condo)** | 2,400 - 2,800 | **极贵**。几乎占掉你税后工资的一半。且需抢房。 |
| **车险 + 油费** | 300 - 500 | 多伦多车险是著名的贵。没车寸步难行。 |
| **手机 + 网费** | 100 - 150 | 全球最贵的电信资费之一。 |
| **超市买菜** | 400 - 600 | 物价在 2024 年涨幅明显。 |
| **税后到手 (75k年薪)** | **~4,400** | |
| **每月剩余** | **~500 - 800** | **除了房租所剩无几** |

**结论：** 在加拿大做单身初级程序员，**存不下钱**。你的生活质量主要体现在“不用加班”和“心情舒畅”，而不是物质丰富。

## 4. 职场与避坑指南

### 4.1 找工渠道
*   **LinkedIn:** 绝对的主流。
*   **Indeed.ca:** 职位很全。
*   **Glassdoor:** 查看薪资和评价。

### 4.2 避坑 (Risks)
1.  **加拿大经验 (Canadian Experience):** 雇主非常看重本地经验。你第一份工作会非常难找，可能需要降薪或去小公司。
2.  **医疗崩坏：** 免费医疗的代价是极长的等待时间。看个专科医生排队 3-6 个月是常态，急诊等 8 小时很正常。
3.  **气候：** 多伦多的冬天从 11 月持续到 4 月。如果你讨厌铲雪和零下 20 度的寒风，你会很抑郁。

## 5. 针对你的行动建议
鉴于你目前分数不够直接 EE 捞人：
1.  **上策 (留学):** 申请 BC 省或安省的 CS Master。这是最稳的，还能解决“本地经验”和“人脉”问题。
2.  **中策 (熬年限):** 在国内大厂（外企更好）熬到 3 年经验，同时把雅思刷到 8777，那时你的 EE 分数才有竞争力。
3.  **下策 (LMIA):** 花钱找中介买雇主担保（LMIA 加 50 分）。**极度不推荐**，风险高，且容易被骗。
