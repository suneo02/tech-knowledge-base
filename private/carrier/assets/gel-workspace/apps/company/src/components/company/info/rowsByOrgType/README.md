# 按组织类型分类的企业信息展示组件

根据不同组织类型的企业信息展示规范，提供定制化的企业基本信息展示组件。

## 目录结构

```
rowsByOrgType/
├── CO.tsx      # 公司(Corporation)企业信息展示组件
├── FCP.tsx     # 外商合资企业信息展示组件
├── FPC.tsx     # 外商独资企业信息展示组件
├── GOV.tsx     # 政府机构信息展示组件
├── handle.ts   # 统一处理入口，根据组织类型选择对应组件
├── IIP.tsx     # 个体工商户信息展示组件
├── NGO.tsx     # 非政府组织信息展示组件
├── OE.tsx      # 其他企业信息展示组件
├── PE.tsx      # 私营企业信息展示组件
├── SOE.tsx     # 国有企业信息展示组件
└── SPE.tsx     # 合伙企业信息展示组件
```

## 关键文件说明

| 文件 | 作用 |
|------|------|
| `handle.ts` | 根据企业组织类型选择对应的信息展示组件，提供统一入口 |
| `GOV.tsx` | 政府机构信息展示布局，包含单位名称、批准机构、机构状态等 |
| `SOE.tsx` | 国有企业信息展示布局，包含企业基本信息、股权结构等 |
| `PE.tsx` | 私营企业信息展示布局，包含企业基本信息、股东信息等 |

## 依赖关系

- **上游**: 企业基本信息类型(`../handle`)、企业组织类型定义
- **下游**: 企业信息展示页面
- **协作**: 通用企业信息行组件(`../rowsCommon`)、国际化工具(`@/utils/intl`)
