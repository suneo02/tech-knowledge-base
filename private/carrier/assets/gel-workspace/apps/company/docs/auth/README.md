# 权限与用户模块文档

本目录包含企业库应用中权限管理和用户相关功能的设计文档。

## 目录结构

- [membership-permissions-interaction](./membership-permissions-interaction.md) - 会员权限与用户交互设计文档

## 相关组件

权限与用户模块主要包含以下组件：

1. **VipModuleNew** - 会员购买模块，提供VIP、SVIP和企业套餐的购买选项
2. **VipForbidden** - 禁止销售提示组件，针对海外用户或特定终端类型
3. **用户协议组件** - UserAgreementBtn 和 PrivacyPolicyBtn
4. **发票样例组件** - InvoiceSample

## 权限体系

企业库采用三级会员权限体系：

1. **VIP** - 基础付费会员，年费398元
2. **SVIP** - 高级付费会员，年费1980元，享有更多高级功能
3. **企业套餐(EP)** - 定制化企业解决方案，需联系客户经理

## 特殊功能

- **活动优惠** - 特定时间段内提供买一年送3个月等优惠活动
- **海外限制** - 部分功能对海外用户有限制
- **终端限制** - 某些终端类型可能不支持购买功能
