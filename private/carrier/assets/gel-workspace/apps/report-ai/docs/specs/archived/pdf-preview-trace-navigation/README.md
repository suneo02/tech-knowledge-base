## 任务背景

当前 PDF 预览功能已支持基础的文件预览和页码跳转，但在处理包含多个章节关联的 trace 数据时，缺少快速导航能力。用户需要：

1. 当文件关联多个章节时，能够快速跳转到不同章节对应的页码位置
2. 当从外部点击引用时，能够直接定位到对应的 trace 数据页码
