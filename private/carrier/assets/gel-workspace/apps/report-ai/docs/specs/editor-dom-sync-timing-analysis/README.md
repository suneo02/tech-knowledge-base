## 快速导航

- **核心问题**：用户每次输入都会触发全量 DOM 扫描，大文档场景下可能导致卡顿
- **推荐方案**：智能变更检测 + 延迟执行组合
- **预期效果**：减少 90% 的不必要同步，提升大文档编辑性能
