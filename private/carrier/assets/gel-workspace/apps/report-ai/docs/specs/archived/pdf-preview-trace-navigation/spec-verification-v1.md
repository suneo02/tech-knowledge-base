# PDF 预览 Trace 数据导航 - 验收文档 v1

> 回链：[任务概览](./README.md)

## 核心用例

### UC-1: 多章节文件

**前置**：`position.length > 1`，章节 2/5/8，页码 10/25/40

**验收**：

- [ ] 显示 QuickJumper，3 个章节标签
- [ ] PDF 从第 1 页开始
- [ ] 点击"第五章"跳转到第 25 页

### UC-2: 单章节文件

**前置**：`position.length === 1`，章节 3，页码 15

**验收**：

- [ ] 显示 QuickJumper，1 个章节标签
- [ ] PDF 从第 1 页开始
- [ ] 点击标签跳转到第 15 页

### UC-3: 外部引用跳转

**前置**：点击引用标记，`initialPage=20`

**验收**：

- [ ] 不显示 QuickJumper
- [ ] PDF 直接定位到第 20 页

### UC-4: 无 Trace 数据

**前置**：`position` 为空

**验收**：

- [ ] 不显示 QuickJumper
- [ ] PDF 从第 1 页开始

### UC-5: 部分章节无位置

**前置**：`position = [pos1, undefined, pos3]`

**验收**：

- [ ] 仅显示 2 个章节标签
- [ ] 跳过 undefined 章节

### UC-6: 章节名称

**验收**：

- [ ] 有 chapterMap 显示章节标题
- [ ] 无 chapterMap 显示"章节 X"

## 边界情况

| 场景                 | 预期             | 状态 |
| -------------------- | ---------------- | ---- |
| `startPage` 超出范围 | 跳转到最后一页   | [ ]  |
| 章节标签 >10 个      | 横向滚动         | [ ]  |
| 快速连续点击         | 流畅无卡顿       | [ ]  |
| PDF 加载中点击       | 等待加载后跳转   | [ ]  |
| `initialPage` 优先级 | 最高，忽略 trace | [ ]  |
