# 首页模块

## 概述

首页模块是 Report AI 的入口页面，提供报告列表、模板选择、新建报告等功能

## 功能特性

- **报告列表**: 展示用户创建的报告列表
- **模板选择**: 提供多种报告模板供用户选择
- **新建报告**: 快速创建新报告
- **搜索过滤**: 支持报告搜索和筛选
