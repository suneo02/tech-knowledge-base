## 二、职业定位与核心诉求

### 2.1 职业定位

- 短期定位（1–3 年）：
  在前端开发方向打牢基础，成长为一名**独立负责中小型模块的前端工程师（Junior → Mid）**，在团队中能承担从需求理解、技术选型到上线运维的完整闭环。

- 中期定位（3–5 年）：
  向**前端工程化 / 前端架构 / 高级前端工程师**方向发展，具备：

  - 对复杂前端项目的整体把控能力
  - 对性能优化、工程化体系（构建、发布、监控）的深入理解
  - 在团队中承担技术推动和 code review 的角色

- 长期定位（5 年及以上）：
  能够在**海外或国际化团队**中工作，成为：

  - 能用英语完成沟通、文档和技术分享的工程师
  - 在某一细分方向有个人品牌（如：工程化、可视化、性能优化、富文本编辑器等）

### 2.2 核心诉求排序

1. 中长期希望拥有**海外工作或长期居住的选择**
2. 不放弃**技术成长和职业天花板**
3. 控制好**经济风险**，不希望为此背上超出能力范围的债务

## 三、现状评估

### 3.1 能力优势

- 有完整的计算机 / 前端基础，大学时期学过较多算法，对基础概念比较扎实
- 已有【1】年一线前端开发经验，接触过真实业务和公司工程流程
- 对工程化、前端工具链等有兴趣，愿意深入理解底层原理
- 自我驱动较强，会主动思考长期发展（如：海外工作、深造路径）

### 3.2 当前短板

- 工作年限偏短，在国际市场上仍然属于 Junior，竞争力有限
- 英文口语与听力不足以支撑高压的全英文面试与日常工作沟通
- 海外经历为零：无海外学历、实习或工作背景
- 对各国留学 / 工作签证政策、实际成本了解不够具体，多停留在“感觉会很贵、很难”的阶段

### 3.3 外部环境简要分析

- 海外初级前端岗位竞争激烈，本地候选人多，对签证敏感
- 海外 CS / IT 硕士项目整体费用偏高，但不同国家、不同学校差异很大
- 远程工作与国际协作机会增多，如能提升英文和技术影响力，未来存在不通过读硕而接触海外团队的可能
