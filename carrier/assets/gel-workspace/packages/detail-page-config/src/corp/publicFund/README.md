# 公募基金配置

公募基金相关信息展示的配置模块，定义了公募基金数据展示结构，包括基金规模、旗下基金等信息的展示配置。

## 目录结构

```
publicFund/
├── FundScale.json                    # 基金规模信息配置
├── UnderlyingFund.json               # 旗下基金信息配置
├── index.json                        # 公募基金配置文件
└── index.ts                          # 模块导出文件
```

## 关键文件说明

- **`index.ts`**: 模块核心导出文件，定义了公募基金配置的导出接口，使用 `validateReportDetailTableJson` 进行表格配置验证。
- **`index.json`**: 公募基金数据展示的核心配置文件，定义了公募基金信息的整体展示结构。
- **`FundScale.json`**: 基金规模信息配置，展示公募基金的管理规模数据。
- **`UnderlyingFund.json`**: 旗下基金信息配置，使用表格形式展示公募基金旗下的各类基金产品。

## 依赖示意

```
publicFund/
├── 依赖 gel-types (ReportDetailTableJson)
├── 依赖 @/validation (validateReportDetailTableJson)
└── 被企业详情页公募基金模块引用
    └── ../index.ts
```

## 配置特点

- **规模展示**: 重点关注公募基金的管理规模数据
- **产品结构**: 详细展示旗下基金产品的构成情况
- **表格形式**: 核心数据使用表格结构清晰展示
- **标准化验证**: 配置文件经过统一验证确保数据正确性

## 应用场景

- **基金投资**: 展示企业参与的公募基金投资情况
- **资产管理**: 了解企业通过公募基金进行的资产配置
- **投资分析**: 分析企业公募基金投资策略和效果
- **规模评估**: 评估企业基金管理业务的规模和实力

## 数据维度

- **规模维度**: 基金管理规模和增长趋势
- **产品维度**: 旗下基金产品的类型和分布
- **收益维度**: 基金产品的收益表现
- **风险维度**: 基金产品的风险等级分布
