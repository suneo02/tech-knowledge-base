# 私募基金配置

私募基金相关信息展示的配置模块，定义了私募基金数据展示结构和布局。

## 目录结构

```
privateFund/
├── index.json                        # 私募基金配置文件
└── index.ts                          # 模块导出文件
```

## 关键文件说明

- **`index.ts`**: 模块核心导出文件，定义了私募基金配置的导出接口。
- **`index.json`**: 私募基金数据展示的核心配置文件，定义了私募基金信息的展示结构。

## 依赖示意

```
privateFund/
├── 依赖 gel-types
└── 被企业详情页私募基金模块引用
    └── ../index.ts
```

## 配置特点

- **简洁结构**: 采用简洁的配置结构，专注私募基金核心信息展示
- **模块化**: 独立模块便于维护和扩展
- **标准化**: 遵循企业详情页配置的统一标准

## 应用场景

- **基金投资**: 展示企业参与的私募基金投资情况
- **资产管理**: 了解企业通过私募基金进行的资产配置
- **投资分析**: 分析企业私募基金投资策略和效果
