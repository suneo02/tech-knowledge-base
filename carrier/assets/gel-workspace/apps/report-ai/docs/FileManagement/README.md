# 文件管理模块

## 概述

文件管理模块负责管理用户上传的文件和企业资料，提供文件列表、上传、删除等功能

## 功能特性

- **文件列表**: 展示用户上传的所有文件
- **批量上传**: 支持批量上传文件
- **文件分类**: 按类型、时间等维度分类
- **企业关联**: 将文件关联到特定企业
