# 大纲视图模块

## 概述

大纲视图模块负责在报告详情页右侧展示报告大纲，提供章节导航和进度指示功能

## 功能特性

- **章节导航**: 点击章节快速定位到对应内容
- **进度指示**: 显示各章节的生成进度
- **层级编号**: 自动生成章节层级编号
- **展开折叠**: 支持章节树的展开和折叠
