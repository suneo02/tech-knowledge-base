# 按国家/地区分类的企业信息展示组件

根据不同国家/地区的企业信息展示规范，提供定制化的企业基本信息展示组件。

## 目录结构

```
rowsByNation/
├── canada.tsx          # 加拿大企业信息展示组件
├── england.tsx         # 英国企业信息展示组件
├── france.tsx          # 法国企业信息展示组件
├── germany.tsx         # 德国企业信息展示组件
├── handle.tsx          # 统一处理入口，根据地区选择对应组件
├── india.tsx           # 印度企业信息展示组件
├── italy.tsx           # 意大利企业信息展示组件
├── japan.tsx           # 日本企业信息展示组件
├── korea.tsx           # 韩国企业信息展示组件
├── lux.tsx             # 卢森堡企业信息展示组件
├── malaysia.tsx        # 马来西亚企业信息展示组件
├── nzl.tsx             # 新西兰企业信息展示组件
├── other.tsx           # 其他国家/地区企业信息展示组件
├── russia.tsx          # 俄罗斯企业信息展示组件
├── singapore.tsx       # 新加坡企业信息展示组件
├── tha.tsx             # 泰国企业信息展示组件
└── vie.tsx             # 越南企业信息展示组件
```

## 关键文件说明

| 文件 | 作用 |
|------|------|
| `handle.tsx` | 根据企业所属地区选择对应的信息展示组件，提供统一入口 |
| `canada.tsx` | 加拿大企业信息展示布局，包含企业名称、注册号、注册地址等 |
| `singapore.tsx` | 新加坡企业信息展示布局，包含数据来源信息 |
| `japan.tsx` | 日本企业信息展示布局，包含国税厅数据来源 |

## 依赖关系

- **上游**: 企业地区类型定义(`@/handle/corp/corpArea`)、企业基本信息类型(`../handle`)
- **下游**: 企业信息展示页面
- **协作**: 通用企业信息行组件(`../rowsCommon`)、国际化工具(`@/utils/intl`)
