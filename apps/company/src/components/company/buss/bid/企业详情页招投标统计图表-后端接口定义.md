# 企业详情页招投标统计图表-后端接口定义

## 趋势分析

`/Wind.WFC.Enterprise.Web/Enterprise/gel/detail/company/tenderTrendAnalyseCompany/{companyCode}`

### 入参

| 参数名      | 类型   | 默认值               | 备注                                                         |
| ----------- | ------ | -------------------- | ------------------------------------------------------------ |
| companyCode | String | 无                   | 万得机构统一编码，取自F1_0088                                |
| interval    | String | 无                   | 表示统计的每段时间区间大小，month\|quarter\|year 中的一种，分别表示月、季、年 |
| startTime   | String | 依interval不同而不同 | 统计开始日期，格式yyyy--MM--dd，不传表示使用默认值。默认值：interval为month时，默认取去年1月1日；interval为quarter时，默认取年份为当前年减4的1月1日；interval为year时，默认取""，表示取所有数据 |
| endTime     | String | 当前日期             | 统计截止日期，格式yyyy--MM--dd                               |

入参样例

```json
{
    "companyCode":"1047934153",
    "interval":"month"
}
```

### 出参

| 参数名      | 类型   | 备注                                                 |
| ----------- | ------ | ---------------------------------------------------- |
| startTime   | String | 统计区间开始时间，格式yyyy--MM--dd                   |
| interval    | String | 表示统计的时间区间大小，month\|quarter\|year中的一种 |
| bidCount    | int    | 该统计区间内中标次数                                 |
| notBidCount | int    | 该统计区间内未中标次数                               |
| totalMoney  | double | 该统计区间内中标总金额                               |

出参样例

```json
[
    {
        "startTime":"2024-01-01",
        "interval":"month",
        "bidCount":12,
        "notBidCount":34,
        "totalMoney":123.45
    },
    {
        "startTime":"2024-02-01",
        "interval":"month",
        "bidCount":56,
        "notBidCount":78,
        "totalMoney":567.89
    }
]
```



## 金额分析

`/Wind.WFC.Enterprise.Web/Enterprise/gel/detail/company/tenderMoneyAnalyseCompany/{companyCode}`

### 入参

| 参数名      | 类型   | 默认值               | 备注                                                         |
| ----------- | ------ | -------------------- | ------------------------------------------------------------ |
| companyCode | String | 无                   | 万得机构统一编码，取自F1_0088                                |
| interval    | String | 无                   | 表示统计的每段时间区间大小，month\|quarter\|year 中的一种，分别表示月、季、年 |
| startTime   | String | 依interval不同而不同 | 统计开始日期，格式yyyy--MM--dd，不传表示使用默认值。默认值：interval为month时，默认取去年1月1日；interval为quarter时，默认取年份为当前年减4的1月1日；interval为year时，默认取""，表示取所有数据 |
| endTime     | String | 当前日期             | 统计截止日期，格式yyyy--MM--dd                               |

入参样例

```json
{
    "companyCode":"1047934153",
    "interval":"month"
}
```

### 出参

| 参数名           | 类型   | 备注                                                 |
| ---------------- | ------ | ---------------------------------------------------- |
| startTime        | String | 统计区间开始时间，格式yyyy--MM--dd                   |
| interval         | String | 表示统计的时间区间大小，month\|quarter\|year中的一种 |
| maxMoney         | double | 该统计区间内最高中标金额                             |
| minMoney         | double | 该统计区间内最低中标金额                             |
| midMoney         | double | 该统计区间内中标金额的中位数                         |
| lowQuarterMoney  | double | 该统计区间内中标金额的下四分之一位数                 |
| highQuarterMoney | double | 该统计区间内中标金额的上四分之一位数                 |

出参样例

```json
[
    {
        "startTime":"2024-01-01",
        "interval":"month",
        "maxMoney":12.34,
        "minMoney":123.56,
        "midMoney":88.8,
        "lowQuarterMoney":33.3,
        "highQuarterMoney":99.9,
    },
    {
        "startTime":"2024-02-01",
        "interval":"month",
        "maxMoney":11.11,
        "minMoney":222.22,
        "midMoney":88.88,
        "lowQuarterMoney":33.33,
        "highQuarterMoney":99.99,
    },
]
```

## 标的物分析

`/Wind.WFC.Enterprise.Web/Enterprise/gel/detail/company/tenderSubjectAnalyseCompany/{companyCode}`

### 入参

| 参数名      | 类型   | 默认值 | 备注                          |
| ----------- | ------ | ------ | ----------------------------- |
| companyCode | String | 无     | 万得机构统一编码，取自F1_0088 |

入参样例

```json
{
    "companyCode":"1047934153"
}
```

### 出参

| 参数名  | 类型   | 备注               |
| ------- | ------ | ------------------ |
| subject | String | 标的物名称         |
| count   | int    | 该地区中标项目个数 |

出参样例

```json
[
    {
        "subject":"030101",
        "count":"17"
    },
    {
        "subject":"030102",
        "count":"15"
    }
]
```



## 地区分布

`/Wind.WFC.Enterprise.Web/Enterprise/gel/detail/company/tenderAreaAnalyseCompany/{companyCode}`

### 入参

| 参数名       | 类型   | 默认值     | 备注                                                         |
| ------------ | ------ | ---------- | ------------------------------------------------------------ |
| companyCode  | String | 无         | 万得机构统一编码，取自F1_0088                                |
| provinceCode | String | 无         | 指定要统计的省份的地区代码，不传或传空则表示统计所有省份     |
| startTime    | String | 去年1月1日 | 统计开始日期，格式yyyy--MM--dd，不传表示使用默认值。默认值为去年1月1日 |
| endTime      | String | 当前日期   | 统计截止日期，格式yyyy--MM--dd                               |

入参样例

```json
{
    "companyCode":"1047934153",
    "provinceCode":"030101"
}
```

### 出参

| 参数名   | 类型   | 备注                                                         |
| -------- | ------ | ------------------------------------------------------------ |
| areaCode | String | 地区代码，按省份聚合时返回各省份代码，按市聚合时返回各市代码 |
| bidCount | int    | 该地区中标项目个数                                           |

出参样例

```json
[
    {
        "areaCode":"030101",
        "bidCount":"17"
    },
    {
        "areaCode":"030102",
        "bidCount":"15"
    }
]
```
