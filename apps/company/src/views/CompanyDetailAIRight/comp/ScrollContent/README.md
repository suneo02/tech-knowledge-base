# ScrollContent - 滚动容器组件

提供统一的滚动区域容器，用于企业详情页的内容滚动。

## 目录树

```
ScrollContent/
├── index.tsx           # 组件实现
├── index.module.less   # 组件样式
└── README.md           # 文档说明
```

## 关键文件说明

| 文件 | 作用 |
|------|------|
| **index.tsx** | 主组件，提供统一的滚动容器和滚动事件处理 |
| **index.module.less** | 组件样式文件，定义滚动容器的样式 |

## 依赖示意

```
ScrollContent
└─> React (React 基础库)
```

- **上游依赖**：React
- **下游使用**：CompanyDetailAIRight 页面
