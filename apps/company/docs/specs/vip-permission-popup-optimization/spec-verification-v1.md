# 企业库会员权限弹窗优化 - 测试与验收

[← 返回任务概览](/apps/company/docs/specs/vip-permission-popup-optimization/README.md)

## 验收标准

### 功能验收

#### 1. 产品文案与展示
- [ ] **第三方提示**：弹窗头部不显示"该数据由第三方提供"文案
- [ ] **产品名称**：第三个产品名称显示为"营销版（百分企业）"
- [ ] **说明文案**：营销版说明显示为"联系客户经理咨询采购"
- [ ] **权限展示**：
  - [ ] 普通/VIP用户展示三个选项（VIP/SVIP/营销版）
  - [ ] 仅营销版权限模块只展示营销版选项

#### 2. 服务链接跳转
- [ ] **营销版跳转**：选中营销版时，点击"查看全部权限和价格"跳转至 `/baifenqi/versionPrice`（或确认后的链接）
- [ ] **VIP/SVIP跳转**：选中 VIP/SVIP 时，点击跳转链接保持原有逻辑（`/versionPrice`）

#### 3. 联系接口交互
- [ ] **接口调用**：选中营销版点击"立即联系"，调用百分模块接口（`baifenqi/operation/post/purchaseIntention`）
- [ ] **成功反馈**：接口调用成功后，显示全局提示"专属客户经理已收到开通需求，将在一个工作日内同您联系"
- [ ] **失败反馈**：接口调用失败时，显示相应的错误提示

#### 4. 遮罩策略（待确认）
- [ ] **表格遮罩**：在指定表格模块触发权限弹窗时，表格内容被遮罩但列头保持可见
- [ ] **其他遮罩**：非表格模块保持原有全覆盖遮罩样式

### 回归测试

- [ ] **VIP购买流程**：选择 VIP -> 立即支付 -> 跳转支付页流程正常
- [ ] **SVIP购买流程**：选择 SVIP -> 立即支付 -> 跳转支付页流程正常
- [ ] **协议勾选**：未勾选协议点击按钮应提示并拦截
- [ ] **海外用户**：海外用户（非 SVIP）访问受限功能应显示 `VipForbidden` 组件而非购买弹窗

## 测试数据

| 场景 | 用户状态 | 预期结果 |
| ---- | -------- | -------- |
| 普通用户访问 | 无权限 | 显示三合一弹窗 |
| VIP用户访问SVIP功能 | VIP | 显示三合一弹窗 |
| 海外用户访问 | 无权限 | 显示 Forbidden 提示 |
| 营销版模块访问 | 无权限 | 仅显示营销版选项 |

- [ ] 检查是否有硬编码的旧文案残留
- [ ] 检查百分接口是否需要鉴权或其他 Header 参数
- [ ] 检查遮罩样式在不同屏幕尺寸下的兼容性
