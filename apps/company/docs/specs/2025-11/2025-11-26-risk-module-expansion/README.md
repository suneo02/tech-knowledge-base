## 需求概述

### 司法风险新增目录（3项）

1. **股权冻结**：仅展示统计数字，点击蒙版引流
2. **限制出境**：仅展示统计数字，点击蒙版引流
3. **悬赏公告**：仅展示统计数字，点击蒙版引流

### 经营风险新增目录（5项）+ 调整（2项）

**新增模块**：

1. **债务逾期**：仅展示统计数字，点击蒙版引流
2. **商票逾期**：仅展示统计数字，点击蒙版引流
3. **环保信用**：仅展示统计数字，点击蒙版引流
4. **股权质押**：仅展示统计数字，点击蒙版引流
5. **简易注销**：仅展示统计数字，点击蒙版引流

**调整位置**（已存在的模块）：

1. **债券违约**：调整到新位置，改为仅展示统计数字，点击蒙版引流
2. **非标违约**：调整到新位置，改为仅展示统计数字，点击蒙版引流

### 目录顺序要求

**司法风险**：裁判文书 → 立案信息 → 开庭公告 → 送达公告 → 法院公告 → 被执行人 → 失信被执行人 → 终本案件 → 限制高消费 → **股权冻结** → 司法拍卖 → 破产重整 → **悬赏公告** → **限制出境** → 询价评估

**经营风险**：诚信信息 → **债券违约** → **非标违约** → **债务逾期** → **商票逾期** → 欠税信息 → 经营异常 → 严重违法 → **环保信用** → 股权出质 → **股权质押** → 税收违法 → 担保信息 → 知识产权出质 → **简易注销** → 抽查检查
