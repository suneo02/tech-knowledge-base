# 通用组件说明文档

本文档描述了项目中使用的通用组件的简要说明，详细文档请点击对应链接查看。

!!!通用组件不允许包含任何业务组件，业务组件请添加至 `components/biz` 目录下

## 目录

### [Loading 加载组件](./Loading/README.md)

提供统一的加载状态展示组件，支持自定义大小和提示文案，适用于各种加载场景。

### [DynamicComponent 动态组件](./DynamicComponent/README.md)

用于动态加载组件的高阶组件，支持懒加载、加载状态展示和错误处理，可用于优化首屏加载性能。

### [PageTransition 页面过渡组件](./PageTransition/README.md)

提供页面间平滑过渡动画效果，支持前进和后退两种动画方向，可用于模态框、标签页、向导等多场景页面切换。

## 开发规范

- 新增组件时，请在对应组件目录下创建 README.md 文档
- 组件目录结构应包含：组件代码、类型定义、样式文件、测试文件和文档
- 确保提供完整的 TypeScript 类型定义
- 编写必要的单元测试用例
